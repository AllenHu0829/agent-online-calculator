# 在线版计算器测试程序

Agent build 产出，符合「标准 Agent 构建 Skill」三阶段流程。

## 运行

- 本地：用浏览器直接打开 `index.html`，或 `npx serve .` 后访问。
- 部署：可将本目录部署到 Vercel / Netlify / GitHub Pages，获得 Agent 在线地址。

## 测试用例（与 Notion 测试用例 v1 一致）

| ID   | 场景   | 操作           | 预期           |
|------|--------|----------------|----------------|
| TC-01 | 加法   | 输入 2+3，=    | 显示 5         |
| TC-02 | 减法   | 输入 10-4，=   | 显示 6         |
| TC-03 | 乘法   | 输入 3*4，=    | 显示 12        |
| TC-04 | 除法   | 输入 8/2，=    | 显示 4         |
| TC-05 | 除零   | 输入 1/0，=    | 提示错误，不崩溃 |
| TC-06 | 在线访问 | 打开部署 URL | 页面加载且可操作 |

## 仓库说明

本 Agent 对应独立 GitHub 仓库；Notion「Agent build」行中会记录 GitHub 与 Agent 在线地址。
